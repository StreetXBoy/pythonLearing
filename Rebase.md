# git 命令学习
新建mydev分支并上传自己的分支

git checkout -b mydev  新建本地分支

git add -A 添加修改

git commit -m "这是注释:初始化项目" 添加注释

git push -u origin mydev  push并上传远程分支 第一次使用 -u  之后可以直接使用 git push origin!!!
